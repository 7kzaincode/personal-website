# Personal Website — Legacy Version

This repository contains an earlier version of my personal portfolio, built with HTML, CSS, and vanilla JavaScript.

The current portfolio lives at [zk-personal-portfolio](https://github.com/7kzaincode/zk-personal-portfolio) and is deployed at [zk-personal-portfolio.vercel.app](https://zk-personal-portfolio.vercel.app/).

## What this version explored

- Animated multilingual preloader
- Light and dark themes
- Typing and scroll-reveal effects
- Interactive project cards
- Mouse trail and particle effects
- Built-in shuffled lo-fi music player
- Patch-notes modal
- Responsive navigation and contact sections

## Run locally

No package installation or build step is required:

```bash
git clone https://github.com/7kzaincode/personal-website.git
cd personal-website
python -m http.server 8000
```

Open [http://localhost:8000](http://localhost:8000).

## Project structure

```text
index.html              Portfolio content and layout
style.css               Main responsive theme
script.js               Interactions, animation, and music controls
patch_notes.html        Earlier standalone patch-notes page
patchnotesstyle.css     Patch-notes styling
logos/                  Project artwork
music/                  Local audio playlist
```

## Status

This version is retained as a record of an earlier design direction. New portfolio changes should go to [`zk-personal-portfolio`](https://github.com/7kzaincode/zk-personal-portfolio).
